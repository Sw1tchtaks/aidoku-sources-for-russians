# ReadManga (ru.readmanga)

Источник для readmanga (по факту сейчас зеркало `a.zazaza.me`) на движке Grouple.

## Реализация

Логика общая с MintManga / SelfManga / AllHentai — вынесена в [`templates/grouple`](../../templates/grouple). Источник — тонкая обёртка с `DEFAULT_BASE_URL`.

В первой версии:

- Каталог: «Популярное» / «Последние обновления» / «Новинки» через `/list?sortType=...`
- Поиск: `/search/advancedResults?q=...` (без расширенных фильтров пока)
- Список глав: парсится из таблицы `tr.item-row`, `?d=user_hash&mtr=true` подкладывается из inline-скрипта
- Чтение: страницы извлекаются из `rm_h.readerInit(...)` / `rm_h.readerDoInit(...)` ручным сканером (без regex-крейта, чтобы не раздувать .aix)

## Настройки

- **Веб-домен** — Grouple часто меняет основной домен. Сменить можно без обновления источника.

## Известные ограничения (v1)

- Нет фильтров по жанрам/категориям/возрасту/сортировке (filters.json пуст). Заглушка под расширение в v2.
- Лицензированные тайтлы и платные главы у MintManga/AllHentai могут просить логин — в первой версии это ронит чтение с понятным сообщением, авторизации пока нет.
- Дата главы парсится по `dd.MM.yy` / `dd/MM/yy` в UTC midnight; локального TZ нет.
