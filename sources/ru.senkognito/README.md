# Senkognito (ru.senkognito)

NSFW-источник для [senkognito.com](https://senkognito.com) — взрослый twin-сайт Senkuro, тот же GraphQL-бэкенд (`api.senkuro.me`).

Реализован поверх [`templates/senkuro`](../../templates/senkuro) — отличается только `BASE_URL` и пустым списком `EXCLUDE_GENRES` (18+ теги не скрываются).

## Настройки

Те же, что у Senkuro (домен API, английские заголовки). Если у вас уже настроен Senkuro, эти значения для Senkognito хранятся отдельно.

## Иконка

Сейчас используется иконка Senkuro как плейсхолдер — заменю на собственную, как только найду адекватный лого-файл.
