# Senkuro (ru.senkuro)

Источник для [senkuro.com](https://senkuro.com) — современная платформа манги, ранобэ и комиксов на русском.

## Реализация

Senkuro использует **GraphQL API** (`https://api.senkuro.me/graphql`) с Apollo persisted queries. Источник вызывает следующие персистентные операции по их sha256-хэшам:

- `search` — поиск манги по запросу
- `fetchManga` — детали манги по slug
- `fetchMangaChapters` — список глав в ветке (cursor-пагинация)
- `fetchMangaChapter` — страницы главы по slug

Если хэши изменятся на стороне Senkuro (это бывает при обновлении фронтенда), источник перестанет работать — нужно будет их обновить в `src/helpers.rs`.

## Настройки

- **Домен** — резервный URL (если основной недоступен): `senkuro.com` / `senkuro.me`. На API-эндпоинт не влияет, влияет только на формирование URL для deep-link и web-ссылок.
- **Английские заголовки** — показывать английские варианты вместо русских.
- **Изображения в оригинальном качестве** — выкл. использует CDN со сжатием (`shiro.senkuro.net`, q=75). Вкл. — оригинальное качество с `mirai.senkuro.net`.
