# Acomics (ru.acomics)

Источник для [acomics.ru](https://acomics.ru) — русский хостинг авторских веб-комиксов.

## Реализация

Простой HTML-парсер.

- **Каталог**: `GET /comics?skip=N` — 10 тайлов на страницу, селектор `section.serial-card`. Заголовок в `h2.title a`, обложка в `a.cover img[data-real-src]`, описание в `p.about`.
- **Поиск**: `GET /search?keyword=…&skip=N`
- **Карточка**: `GET /~{slug}` — `og:image`, `og:description`, категории.
- **Главы**: на Acomics нет глав в обычном смысле — каждый комикс это серия отдельных «выпусков» (issues), каждый = одна страница. Я объединяю все выпуски в **одну псевдо-главу** «Все страницы», чтобы Aidoku-читалка могла скроллить весь архив подряд.
- **Страницы (выпуски)**: walk `/~{slug}/list?skip=0,5,10,…` — каждый ответ возвращает 5 выпусков с `<img src="/upload/!c/...">`. Источник идёт по странице списка, пока не получит < 5 элементов или не упрётся в кап `ISSUE_LIST_PAGES_CAP=200` (то есть до ≈1000 страниц на комикс). Дедупликация по URL.

## Известные ограничения

- Нет фильтров по жанрам/возрастному рейтингу — `filters.json` пуст.
- Все выпуски сваливаются в одну главу. Для очень длинных комиксов (Three in a Tree = 225, Стандофф 2 ~600) это означает медленный refresh — каждый list-page = 1 HTTP-запрос.
- Webcomic-страницы вертикальные → `Viewer::Webtoon` по умолчанию.
- Status комикса не парсится (нет стабильного индикатора в HTML), всегда `Unknown`.
- Возрастной рейтинг (PG/R) не используется — все комиксы помечены `Safe`.
